# REST API using Golang with JWT tokens and hashed passwords

## Run

To run the program, simply run:

```bash
go run main.go
```

You can then use Postman or any similar software to use the API.

## Endpoints
```bash
/signup POST
```

```bash
/login POST
```

```bash
/protected GET
```
